# Tekken7 Movelist

철권7의 기술 프레임표를 보여줍니다.

줄리아 창 유저라서 줄리아 창을 대상으로 먼저 만들어보았으며, 다른 캐릭터의 프레임표도 차차 추가해나갈 예정입니다.
